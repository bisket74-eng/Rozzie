ROZZIE'S PLAYHOUSE

Files:

- service-worker.js
- icon-192.png
- icon-512.png

For camera and microphone access, open this through HTTPS (for example GitHub Pages), not by double-clicking the file on some phones.

GitHub Pages setup:
1. Create a repository.
2. Upload all five app files to the repository root.
3. Open Settings > Pages.
4. Choose Deploy from a branch, main, /(root), then Save.
5. Open the new HTTPS site on the phone.
6. Use the browser menu and choose Add to Home screen or Install app.

The app does not upload photos or voice recordings. The short voice recording is stored only in the browser's local storage.
